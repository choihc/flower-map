# 꽃 어디

전국 꽃 명소와 축제 정보를 지도 중심으로 탐색하는 서비스입니다.

현재 저장소는 아래 두 제품을 함께 관리하는 구조를 기준으로 시작합니다.

- `apps/mobile`: React Native + Expo 앱
- `apps/admin`: Next.js 어드민
- `docs`: 제품/디자인/개발 가이드

## 추천 개발 순서

1. 모바일 MVP 화면 구조 확정
2. Supabase 스키마 및 API 계약 정의
3. React Native 지도/목록/상세 흐름 구현
4. 어드민 데이터 관리 화면 구현

## MVP 핵심 기능

- 지도에서 꽃 명소 마커 보기
- 꽃 종류별 필터
- 명소 상세 정보
- 길찾기 외부 앱 연결
- 즐겨찾기 저장
- 어드민에서 명소/축제 데이터 검수

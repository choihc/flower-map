# 꽃 어디 화면 구조 초안

## 1. 모바일 앱 핵심 화면

### 1. 스플래시

- 로고
- 시즌성 배경 이미지 또는 컬러 그라데이션
- 앱 진입 로딩

### 2. 홈

- 상단 인사/지역 요약
- "지금 보기 좋은 꽃" 추천 카드 섹션
- "곧 끝나는 축제" 카드 섹션
- 꽃 종류 바로가기 칩
- 인기 지역/에디터 추천 카드
- 하단 탭바

핵심 목적:

- 사용자가 앱에 들어오자마자 현재 시점 기준 추천 명소를 빠르게 고른다.

### 3. 지도 탐색

- 네이버 지도
- 현재 위치 버튼
- 꽃 종류 필터
- 지역 필터
- 지도 하단 선택 명소 카드
- 리스트 보기 전환 버튼

핵심 목적:

- 지도로 주변 또는 특정 지역의 꽃 명소를 탐색한다.

### 4. 명소 리스트

- 선택된 필터 요약
- 카드 리스트
- 정렬: 거리순 / 인기순 / 종료 임박순

핵심 목적:

- 지도보다 리스트가 편한 사용자에게 빠른 비교 수단을 제공한다.

### 5. 필터 바텀시트

- 꽃 종류 선택
- 지역 선택
- 진행 중 축제만 보기
- 이번 주 추천만 보기
- 초기화 / 적용

### 6. 명소 상세

- 대표 이미지
- 명소명 / 꽃 종류 / 시즌 배지
- 축제 기간
- 주소
- 설명
- 주차 / 입장료 / 운영 팁
- 즐겨찾기
- 길찾기

핵심 목적:

- 방문 결정에 필요한 정보를 한 화면에서 제공한다.

### 7. 즐겨찾기

- 저장한 명소 리스트
- 시즌 상태 배지
- 삭제 또는 정렬

### 8. 로그인

- 카카오 로그인
- 구글 로그인
- 게스트 안내

권장 진입 시점:

- 앱 첫 진입에서는 건너뛰기 허용
- 즐겨찾기나 알림 저장 시 로그인 유도

### 9. 내 정보

- 로그인 상태
- 저장한 관심 꽃 종류
- 알림 설정
- 앱 정보 / 문의

## 2. 어드민 핵심 화면

### 1. 로그인

- 관리자 인증

### 2. 대시보드

- 이번 주 노출 중인 명소 수
- 종료 임박 축제 수
- 비공개 대기 데이터 수

### 3. 명소 목록

- 검색
- 꽃 종류 필터
- 노출 상태 필터
- 축제 기간 필터
- 수정 진입

### 4. 명소 등록/수정

- 기본 정보
- 지도 좌표
- 꽃 종류 연결
- 축제 기간
- 썸네일
- 주차 / 입장료 / 설명
- 공개 여부

### 5. 꽃 종류 관리

- 꽃 이름
- 아이콘
- 대표 색상
- 시즌 월 범위
- 활성/비활성
- 정렬 순서

핵심 목적:

- 운영자가 앱 배포 없이 꽃 종류를 확장할 수 있게 한다.

### 6. 사용자 콘텐츠 관리

- 추후 Phase 2 이후 오픈

## 3. 홈 화면 카드 구조 제안

### 섹션 1. 지금 보기 좋은 꽃

- 현재 날짜가 `bloom_start_at`과 `bloom_end_at` 사이인 명소 우선
- 꽃 종류별 대표 카드 노출

### 섹션 2. 곧 끝나는 축제

- `festival_end_at`이 지나지 않은 데이터만 노출
- 종료일 가까운 순 정렬

### 섹션 3. 지역별 추천

- 서울/경기
- 부산/경남
- 전주/남도
- 제주

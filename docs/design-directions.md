# 꽃 어디 디자인 방향 초안

## A. 봄 소풍 감성형

- 키워드: 따뜻함, 산뜻함, 사진 맛집, 주말 나들이
- 컬러: 크림, 코랄, 연두, 벚꽃 핑크
- 적합한 경우: 커플/가족 중심 대중 서비스
- 장점: 친근하고 확산성 좋음
- 주의점: 정보 서비스의 신뢰감이 약해질 수 있음

## B. 지도 탐색 정보형

- 키워드: 빠른 탐색, 명확한 정보, 실용성
- 컬러: 아이보리, 딥그린, 샌드, 오렌지 포인트
- 적합한 경우: 축제/여행 검색 도구로 포지셔닝
- 장점: 사용성이 좋고 어드민/웹으로 확장 쉬움
- 주의점: 브랜드 감성이 약해질 수 있음

## C. 프리미엄 시즌 가이드형

- 키워드: 큐레이션, 에디토리얼, 여행 매거진
- 컬러: 웜화이트, 버건디, 세이지, 골드 베이지
- 적합한 경우: 고급 여행/라이프스타일 브랜드 확장
- 장점: 차별화가 강함
- 주의점: 초기 개발 난이도가 상대적으로 높음

## 추천

MVP는 `B. 지도 탐색 정보형`을 기본으로 두고, 홈과 상세 화면에 `A. 봄 소풍 감성형`의 색감만 일부 섞는 방향이 가장 안정적입니다.

## 현재 선택 방향

- 구조: 정보형
- 인상: 감성적인 봄 시즌 무드
- 화면 톤: 복잡하지 않게, 사진과 카드가 예쁘게 보이는 방식

## 메인 컬러 후보

### 1. Blossom Coral

- `#F26D85`
- 벚꽃과 가장 직접적으로 연결되는 컬러
- 감성적이고 브랜드 인상이 강함
- 너무 넓게 쓰면 피로할 수 있어 포인트 컬러에 적합

### 2. Canola Yellow

- `#F4C542`
- 유채꽃 느낌이 살아나고 시즌감이 분명함
- 홈 추천 카드나 배지에서 생기가 좋음
- 메인 컬러로 쓰면 정보성 UI 대비가 약할 수 있음

### 3. Leaf Green

- `#4D8B55`
- 지도/정보 서비스와 잘 맞고 신뢰감이 높음
- 꽃 컬러를 보조적으로 살리기 쉬움
- 다소 무난해질 수 있어 감성 포인트 컬러 조합이 필요함

### 4. Tulip Peach

- `#F29E7E`
- 따뜻하고 부드러워 카드 UI와 잘 어울림
- 벚꽃/튤립/봄꽃 전체를 포괄하기 좋음
- 정보 서비스의 선명함은 초록 계열보다 낮음

## 추천 조합

가장 안정적인 조합은 아래 두 가지입니다.

### 조합 A

- 메인: `Leaf Green #4D8B55`
- 포인트: `Blossom Coral #F26D85`
- 배경: `Warm Ivory #FFF9F3`

특징:

- 정보형 구조에 가장 잘 맞음
- 꽃 서비스다운 감성 포인트를 살리기 쉬움

### 조합 B

- 메인: `Tulip Peach #F29E7E`
- 포인트: `Leaf Green #4D8B55`
- 배경: `Cream Sand #F7F1E8`

특징:

- 더 따뜻하고 라이프스타일 앱 같은 인상
- 브랜드성은 강하지만 정보 탐색 화면은 조금 더 섬세한 밸런스가 필요함

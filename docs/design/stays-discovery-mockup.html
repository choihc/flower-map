스펙 요약

  • 탭바 도트 — 8pt secondary #F26D85 + 2pt 캔버스 외곽 링. 정적, enter spring 1회.
  • 발견 카드 — 132pt 높이, 28 반경, inkDeep → accentGold 그라디언트, NEW 배지(흰 배경 반전), accentGold CTA pill.
  • 위치 — TopSpotsSection 아래, 기존 SeasonCurationSlot 위치를 그대로 점유.
  • 디스미스 — 통합 boolean hasSeenStays='true'. 탭 진입/카드 탭/X 어느 쪽이든 영구 OFF.

  • 서버 의존 — 0. AsyncStorage 1회 read만.

탭바 도트는 아이콘 프레임 34×34 기준 top:2/right:2에 위치. 외곽 링이 스크롤 그림자 위에서도 도트를 분리시킵니다.

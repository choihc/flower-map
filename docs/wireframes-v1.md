# 꽃 어디 와이어프레임 v1

## 1. 홈 화면

목표:

- 사용자가 앱을 열자마자 지금 가기 좋은 꽃 명소를 고르게 한다.

구성:

```text
+--------------------------------------------------+
| [상단] 꽃 어디                                   |
| 지금 서울은 벚꽃 보기 좋아요                     |
|                                                  |
| [검색처럼 보이는 버튼] 어느 지역 꽃을 찾고 있나요? |
|                                                  |
| [섹션] 지금 보기 좋은 꽃                         |
| +----------------------------------------------+ |
| | 벚꽃 | 이번 주 절정                           | |
| | 여의도 윤중로                                | |
| | 오늘 기준 가장 인기 있는 벚꽃 명소           | |
| | [지도 보기] [길찾기]                         | |
| +----------------------------------------------+ |
| +----------------------------------------------+ |
| | 유채꽃 | 지금 방문 추천                      | |
| | 제주 녹산로                                  | |
| | 노란 들판 포토 스팟                          | |
| | [지도 보기] [길찾기]                         | |
| +----------------------------------------------+ |
|                                                  |
| [섹션] 곧 끝나는 축제                           |
| +----------------------------------------------+ |
| | 진해 군항제                                  | |
| | 4월 7일까지                                   | |
| | 경남 창원시 진해구                           | |
| +----------------------------------------------+ |
|                                                  |
| [섹션] 꽃 종류로 보기                           |
| [벚꽃] [진달래] [튤립] [유채꽃]                  |
|                                                  |
| [섹션] 지역별 추천                              |
| [서울/경기] [부산/경남] [전주/남도] [제주]      |
|                                                  |
|      [홈] [지도] [저장] [내 정보]               |
+--------------------------------------------------+
```

상세 요소:

- 상단 문구는 현재 위치 또는 최근 선택 지역 기준으로 달라진다.
- 첫 카드 섹션은 `현재 날짜가 bloom_start_at~bloom_end_at 사이`인 명소 우선이다.
- 두 번째 섹션은 `festival_end_at >= 오늘`인 데이터만 노출한다.
- 카드마다 대표 꽃 색을 은은하게 반영한다.
- 홈에서 카드 탭 시 상세로 이동하고, `지도 보기`는 지도 화면으로 이동한다.

필수 컴포넌트:

- 시즌 추천 카드
- 종료 임박 축제 카드
- 꽃 종류 칩
- 지역 칩
- 하단 탭바

## 2. 지도 탐색 화면

목표:

- 지역과 꽃 종류를 기준으로 명소를 빠르게 탐색한다.

구성:

```text
+--------------------------------------------------+
| [상단] 지역 선택            [필터] [리스트]      |
|                                                  |
|              네이버 지도 영역                    |
|        o 벚꽃   o 튤립   o 유채꽃 마커           |
|                                                  |
|                            [현재 위치]           |
|                                                  |
| [하단 플로팅 칩] [벚꽃] [진달래] [튤립] [유채꽃] |
|                                                  |
| +----------------------------------------------+ |
| | 여의도 윤중로                                | |
| | 벚꽃 · 서울 영등포구                         | |
| | 이번 주 절정 / 축제 진행 중                  | |
| | [상세 보기] [길찾기]                         | |
| +----------------------------------------------+ |
|                                                  |
|      [홈] [지도] [저장] [내 정보]               |
+--------------------------------------------------+
```

상세 요소:

- 마커 색상은 꽃 종류별 대표 컬러를 사용한다.
- 필터 버튼을 누르면 바텀시트가 열린다.
- 리스트 버튼을 누르면 같은 조건의 리스트 화면으로 전환된다.
- 마커 탭 시 하단 요약 카드가 바뀐다.

필터 조건:

- 꽃 종류
- 지역
- 진행 중 축제만
- 이번 시즌만

## 3. 명소 상세 화면

목표:

- 사용자가 방문 결정을 내릴 수 있도록 필요한 정보를 바로 보여준다.

구성:

```text
+--------------------------------------------------+
| [상단 이미지]                                    |
|                                                  |
| [뒤로]                         [저장 아이콘]     |
|                                                  |
| 여의도 윤중로                                    |
| 벚꽃  |  이번 주 절정                            |
| 서울 영등포구                                    |
|                                                  |
| [길찾기] [지도에서 보기]                         |
|                                                  |
| [섹션] 방문 정보                                 |
| 축제 기간: 2026.04.01 - 2026.04.07               |
| 입장료: 무료                                     |
| 주차: 인근 공영주차장 이용 권장                  |
|                                                  |
| [섹션] 소개                                      |
| 한강을 따라 벚꽃 산책을 즐길 수 있는 대표 명소   |
|                                                  |
| [섹션] 운영 팁                                   |
| 주말 오후 혼잡 / 평일 오전 추천                  |
|                                                  |
| [섹션] 비슷한 꽃 명소                            |
| 여의도공원 / 석촌호수 / 경화역                   |
+--------------------------------------------------+
```

정보 우선순위:

1. 명소명 / 꽃 종류 / 시즌 상태
2. 길찾기 CTA
3. 축제 기간과 방문 정보
4. 설명
5. 비슷한 명소 추천

상태 배지 예시:

- 지금 보기 좋아요
- 이번 주 절정
- 곧 종료

## 4. 즐겨찾기 화면

목표:

- 저장한 명소를 다시 찾고 시즌 상태를 빠르게 확인한다.

구성:

```text
+--------------------------------------------------+
| 저장한 명소                                      |
| 이번 시즌 다시 가볼 만한 곳을 모아봤어요         |
|                                                  |
| [필터] 전체 / 지금 시즌 / 종료 임박              |
|                                                  |
| +----------------------------------------------+ |
| | 여의도 윤중로                                | |
| | 벚꽃 · 이번 주 절정                          | |
| | 서울 영등포구                                | |
| | [길찾기] [삭제]                              | |
| +----------------------------------------------+ |
| +----------------------------------------------+ |
| | 제주 녹산로                                  | |
| | 유채꽃 · 지금 방문 추천                      | |
| | 제주                                          | |
| | [길찾기] [삭제]                              | |
| +----------------------------------------------+ |
|                                                  |
| 로그인하면 기기 변경 시에도 저장 목록이 유지돼요 |
| [로그인하기]                                     |
|                                                  |
|      [홈] [지도] [저장] [내 정보]               |
+--------------------------------------------------+
```

상세 요소:

- 비로그인 상태에서도 로컬 저장은 허용한다.
- 로그인 전에는 하단 고정 영역으로 계정 연동 이점을 안내한다.
- 시즌 배지로 지금 갈 가치가 높은 명소를 먼저 보이게 한다.

## 5. 어드민 꽃 종류 관리 화면

목표:

- 운영자가 앱 배포 없이 꽃 종류를 추가, 수정, 정렬, 비활성화한다.

구성:

```text
+--------------------------------------------------+
| 꽃 종류 관리                       [새 꽃 추가]  |
| 꽃 종류는 앱 홈/지도/필터에 바로 반영됩니다      |
|                                                  |
| 검색: [ 벚꽃                                     ]|
|                                                  |
| +----------------------------------------------+ |
| | 아이콘 | 이름  | 대표색  | 시즌   | 상태 | 수정| |
| |  🌸    | 벚꽃  | #F26D85 | 3-4월  | 활성 | 버튼| |
| |  🌺    | 진달래| #E05A87 | 3-4월  | 활성 | 버튼| |
| |  🌷    | 튤립  | #F29E7E | 4-5월  | 활성 | 버튼| |
| |  🌼    | 유채꽃| #F4C542 | 3-4월  | 활성 | 버튼| |
| +----------------------------------------------+ |
|                                                  |
| [정렬 순서 저장]                                 |
+--------------------------------------------------+
```

생성/수정 폼:

```text
이름(한글)
이름(영문)
슬러그
대표 아이콘
대표 색상
시즌 시작 월
시즌 종료 월
정렬 순서
활성 여부
[저장]
```

운영 규칙:

- 비활성 꽃 종류는 앱 필터와 홈 노출에서 제외한다.
- 이미 연결된 명소가 있으면 삭제 대신 비활성 처리한다.
- 대표 색상은 카드 배경, 배지, 마커 포인트색으로 재사용한다.

## 6. 모바일 공통 UI 방향

레이아웃:

- 정보 구조는 단순하게 간다.
- 홈과 리스트는 카드 중심으로 구성한다.
- 지도는 최대한 넓게 보여준다.

비주얼:

- 배경은 완전 흰색보다 따뜻한 아이보리 계열을 사용한다.
- 카드 모서리는 너무 둥글지 않게, 적당히 부드러운 수준으로 간다.
- 꽃 종류별 포인트색을 쓰되 전체 UI는 차분하게 유지한다.

추천 컬러 조합:

- 메인: `Leaf Green #4D8B55`
- 포인트: `Blossom Coral #F26D85`
- 보조: `Canola Yellow #F4C542`
- 배경: `Warm Ivory #FFF9F3`
- 텍스트: `Forest Ink #243127`

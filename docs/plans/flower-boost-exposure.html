핵심 요약

DB 컬럼 2개 추가 → 어드민 설정 UI → 모바일 정렬 반영 순으로 진행한다. 2개 트랙으로 병렬 구현하며, 각 단계는 TDD(RED→GREEN)로 진행한다.

트랙범위주요 산출물
A DB·어드민apps/web, supabase마이그레이션, 스키마 검증, 폼, 목록 배지
B 모바일apps/mobile정렬 헬퍼, 홈TOP 병합, 검색·목록
핵심 주의: getTopSpots는 홈 TOP과 호캉스 호텔 추천 양쪽에서 쓰인다. 부스트는 홈 전용 신규 함수 getTopSpotsWithBoost로만 적용하고, getTopSpots는 불변으로 유지해 호텔 추천 회귀를 방지한다.
